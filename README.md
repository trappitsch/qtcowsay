# qtcowsay

A simple QtApplication based on [pycowsay](https://github.com/cs01/pycowsay).

## Installation

Clone the project and install it with `pip`:

```bash
git clone https://github.com/trappitsch/qtcowsay.git
cd qtcowsay
pip install .
```

## Usage

```bash
cd src
python -m qtcowsay
```
